🗺️ React Expense Tracker - Build Order List
Step 1: UI එකේ පදනම දැමීම (Static Layout)
අපි මුලින්ම කරන්නේ ඇප් එකේ පෙනුම (Layout) එක හදාගන්න එක. දැනට මේවට ඩේටා දාන්නේ නැහැ, නිකන් HTML/CSS වගේ පෙනෙන්න විතරක් හදනවා.

🏠 Header Component: ඇප් එකේ මාතෘකාව ("Expense Tracker 💰") සහ Reset බටන් එක.

💵 Balance Component: Current Balance, Total Income, සහ Total Expense පෙන්වන කාඩ් 3 (දැනට නිකන් $0.00 වගේ දාලා තියන්නේ).

📋 Transaction List & Item: ලිස්ට් එකක් ඇතුළේ Sample Transaction එකක් (උදා: "Salary - $500") පෙනෙන විදිහට static කාඩ් එකක් හැදීම.

➕ Transaction Form: Description input, Amount input, සහ Type (Income/Expense) තෝරන Dropdown/Radio buttons සහිත ෆෝම් එකේ පෙනුම හැදීම.

Step 2: ෆෝම් එකෙන් ඩේටා ගැනීම (State & Forms)
දැන් අපි ඇප් එකට පණ දෙනවා. යූසර් ටයිප් කරන දේවල් React වලට අල්ලලා දෙන කොටස.
5. ⌨️ Form Inputs (Controlled): Description සහ Amount inputs දෙක Controlled Components බවට පත් කිරීම (useState සහ onChange දාලා).
6. 🔘 Add Transaction Button & Event: බටන් එක ක්ලික් කරාම ෆෝම් එකේ ඩේටා ටික එකතු කරගෙන Object එකක් හදන ලොජික් එක ලිවීම.
7. 📥 Data Array State: අර හදාගන්නා Object එක ප්‍රධාන ලිස්ට් එකට (Transactions Array) එකතු කිරීම.

Step 3: ඩේටා Screen එකේ පෙන්වීම සහ මැකීම (Render & Delete)
🔄 Dynamic List Render: අර Array එක ඇතුළට අලුතින් එකතු වන හැම දේම .map() එකෙන් Screen එකේ ලස්සන කාඩ් විදිහට පෙන්වීම.

🎨 Color Indicators: Income නම් කොළ පාටත්, Expense නම් රතු පාටත් බෝඩර්/අකුරු වැටෙන්න Conditional Rendering දැමීම.

🗑️ Delete Transaction: හැම කාඩ් එකක් ගාවම තියෙන "Delete" බටන් එක ක්ලික් කළාම .filter() එකෙන් ඒක ලිස්ට් එකෙන් අයින් කිරීම.

Step 4: ගණිතමය වැඩ කොටස (Calculations)
📈 Income Calculation: ලිස්ට් එකේ තියෙන ඔක්කොම Income ටික විතරක් එකතු කරලා "Total Income" එක Live හැදීම.

📉 Expense Calculation: ඔක්කොම Expense ටික එකතු කරලා "Total Expense" එක Live හැදීම.

⚖️ Current Balance: Income එකෙන් Expense එක අඩු කරලා "Current Balance" එක Live හැදීම.

Step 5: සෙවීම් සහ පෙරීම් (Search & Filter)
🔍 Search Bar: ටයිප් කරන අකුරු අනුව ලිස්ට් එකේ තියෙන දේවල් ක්ෂණිකව Filter වීම.

🎯 Filter Tabs: All / Income / Expense කියන බටන් 3 ක්ලික් කරද්දී අදාල Transactions විතරක් වෙන් වී පෙනීම.

Step 6: දත්ත සුරැකීම සහ අමතර විශේෂාංග (Advanced Features)
💾 Local Storage: ඇප් එක Close කරලා ආයෙ ඕපන් කරත්, ඩේටා ටික මැකෙන්නේ නැතුව බ්‍රවුසර් එකේ සේව් වී තිබීම.

✏️ Edit/Update Transaction: දාපු එකක් වැරදුණොත් ඒක ආයෙ වෙනස් කරන්න පුළුවන් ලොජික් එක (මේක තමයි ටිකක්ම අමාරු කෑල්ල).

🌙 Dark Mode & Export CSV (Optional): ඇප් එක ලස්සන කරන්න ඩාර්ක් මෝඩ් එක සහ ඩේටා ටික Excel ශීට් එකකට ගන්නා ක්‍රමය.

🚀 දැන් අපේ තත්ත්වය:
අපි දැනට ඉන්නේ Step 1 හි 1 වෙනි පියවරේ (Header Component).

අර npm install එක ඉවර වුණු ගමන්, ඔයා Header.jsx එක ලියලා එවන්න මචං. අපි එතන ඉඳන් මේ ලිස්ට් එකේ පල්ලෙහාටම පිළිවෙළට යමු! 🦾🔥
